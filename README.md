# 📊 Meta Graph API - Post Performance Analyzer

A professional, robust **Spring Boot** backend application designed to fetch, persist, and analyze social media post performance using the **Meta Graph API**. This project tracks engagement metrics (likes, comments) across the latest 20 posts, computes analytical trends based on the days of the week, and discovers high-performing content strategy insights.

---

## 🚀 Key Features & Tech Stack

* **Java 17 & Spring Boot 3.5.4**
* **Spring Data JPA & PostgreSQL** (With targeted database indexing for optimized query speeds)
* **Spring RestClient** (Modern, fluid API wrapper utilized for clean HTTP communication with Meta endpoints)
* **Dotenv Java** (Secure environment configuration managing sensitive data without hardcoding API credentials)
* **Spring Scheduler** (Automated background synchronization tasks)
* **Global Exception Handling Architecture** & **Lombok** ---
